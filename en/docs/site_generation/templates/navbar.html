    Navbar 露

    Navbar can have a search box and buttons.

    To enable search box just set ENABLE_SEARCH setting. Similar to manual sidebar you can provide a navbar.json with the following content:

    [
  {
    "caption": "Github",
    "url": "https://github.com",
    "class": "btn-primary"
  }
]
