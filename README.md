# Docurry

A spicy documentation site generator.

- [ ] Write content in pure markdown.
- [ ] Plain html output. Static. No routers
- [ ] Auto generate sidebar
- [ ] Multi-language support
- [ ] SEO first
- [ ] Mobile first
- [ ] Auto generate search
- [ ] Auto generate landing page.
